Murmur's ICEv3.5 userConnected/userDisconnected callbacks-notifications adder and listener.
Works only with a Murmur that supports ICE's version 3.5 protocol (for now) and runs at localhost.

Build program using: 
`make`

Run program using: 
`./murmur_callbacks`

Enjoy!
